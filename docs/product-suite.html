Our Product Suite

TrusteD-V offers a comprehensive suite of secure, Rust-based software components, tools, and solutions for RISC-V embedded systems, from low-level bootloaders to high-level IDEs.

Secure Bootloader (rustBoot)
Ensure system integrity from power-on with our robust, Rust-based secure bootloader.
Trusted Hardware Abstraction Layer (HAL)
Simplify hardware interaction with a secure and efficient HAL for RISC-V processors.
Rust Compiler with Pliron & Cranelift
Optimized Rust compiler toolchain for RISC-V, leveraging advanced backends for performance.
Secure Microkernel & OS Integrations
Build reliable embedded systems with our secure microkernel and Tock OS/Linux integrations.
Integrated SDKs for Heterogeneous Chips
Comprehensive SDKs for various RISC-V processors, including AI accelerators (TPU/NPU).
Vertically Integrated IDE (SLM-based)
A powerful, web-based IDE designed for seamless Rust and RISC-V development.
Debugging & Flashing Toolchains (Probe.rs)
Advanced tools for efficient debugging and secure flashing of RISC-V devices.
Certified Rust Toolchains (ISO 26262)
Achieve functional safety compliance with our certified Rust toolchains for automotive applications.
Peripheral Access Crates
Rust crates providing safe and idiomatic access to RISC-V peripherals.
Trusted Accelerator APIs (GPU/VPU)
Secure APIs for integrating and utilizing hardware accelerators like GPUs and VPUs.
Secure Hardware Security Module (HSM) Integration
Seamless integration with HSMs for enhanced cryptographic operations and secure key storage.
Software Testing Toolchain
Comprehensive tools for automated testing, verification, and validation of embedded software.
Shakti SDK
The official SDK for the Shakti processor family, enabling rapid development.
LLVM Integration
Leveraging the LLVM infrastructure for advanced compiler optimizations and code generation.
Full-stack Secure Embedded System Software
End-to-end secure software solutions from bootloader to application layer.
Custom SDKs for RISC-V Processors
Tailored SDKs designed to meet the unique requirements of your specific RISC-V hardware.
Secure Software Stacks for TPUs, NPUs, and AI Accelerators
Specialized software solutions ensuring security and performance for AI-driven embedded systems.
